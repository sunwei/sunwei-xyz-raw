MDFriday

我是 MDFriday 的创始人，这是一个让你将 Markdown 文件转换为网站 的工具——完全离线运行。
经过近三年的努力，我重写了世界上最快的静态网站生成器之一 Hugo 的核心，使 MDFriday 能够支持 500+ Hugo 主题。

目前，我们支持 Obsidian，这意味着你可以一键将 Markdown 笔记转换为多语言网站——全部在本地机器上完成。
这只是开始！我们很快将支持更多平台，帮助世界各地的创作者轻松展示他们的作品。

使用 MDFriday，你可以：

  • 直接从 Markdown 笔记构建网站
  • 即时预览网站，享受闪电般快速的本地更新
  • 从 500+ 精美主题中选择
  • 以多种方式部署网站
